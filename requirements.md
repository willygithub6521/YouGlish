# YouTube 發音搜尋平台 - 需求文件

## 1. 專案概述

建立一個類似 YouGlish 的英語發音學習平台，讓用戶可以搜尋英文單字或短語，並從真實的 YouTube 影片中找到該單字的發音範例。

## 2. 核心功能需求

### 2.1 搜尋功能
- 用戶可以輸入英文單字或短語進行搜尋
- 支援模糊搜尋和完全匹配
- 搜尋結果按相關性排序
- 顯示找到的影片數量

### 2.2 口音篩選
- 支援以下口音篩選：
  - 全部（All）
  - 美國英語（US）
  - 英國英語（UK）
  - 澳洲英語（AU）
  - 加拿大英語（CA）
  - 其他英語系國家

### 2.3 影片播放
- 嵌入 YouTube 影片播放器
- 自動跳轉到單字出現的時間點
- 顯示當前播放的字幕
- 支援播放控制（播放、暫停、上一個、下一個）

### 2.4 字幕顯示
- 顯示包含搜尋單字的完整句子
- 高亮顯示搜尋的單字
- 顯示前後文上下文

### 2.5 連續播放
- 自動播放下一個包含該單字的影片片段
- 顯示當前是第幾個結果（例如：1/50）
- 支援手動切換到上一個或下一個結果

## 3. 技術需求

### 3.1 前端
- React 18+ with TypeScript
- Tailwind CSS for styling
- React Query for data fetching
- YouTube IFrame Player API

### 3.2 後端
- Node.js 18+ with Express
- TypeScript
- RESTful API design

### 3.3 資料庫
- PostgreSQL（影片元數據、字幕）
- Elasticsearch（全文搜尋）
- Redis（快取）

### 3.4 外部 API
- YouTube Data API v3
- YouTube Transcript API

## 4. 非功能需求

### 4.1 效能
- 搜尋回應時間 < 2 秒
- 影片載入時間 < 3 秒
- 支援至少 1000 個並發用戶

### 4.2 可用性
- 響應式設計，支援桌面和行動裝置
- 直覺的使用者介面
- 鍵盤快捷鍵支援

### 4.3 可擴展性
- 模組化架構
- 易於新增更多語言支援
- 易於新增更多影片來源

## 5. 用戶故事

### 5.1 基本搜尋
作為一個英語學習者，我想要搜尋一個單字，以便聽到它在真實情境中的發音。

### 5.2 口音學習
作為一個英語學習者，我想要篩選特定口音的發音範例，以便學習我感興趣的口音。

### 5.3 上下文學習
作為一個英語學習者，我想要看到單字在句子中的使用方式，以便理解它的實際用法。

### 5.4 多範例學習
作為一個英語學習者，我想要連續播放多個範例，以便更好地掌握單字的發音變化。
